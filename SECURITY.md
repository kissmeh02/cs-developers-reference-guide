# Security policy

This repository is a static documentation site (HTML, CSS, and client-side JavaScript). It does not run a server or handle user accounts in this repo.

## Reporting a concern

If you discover a security issue that applies to this project (for example, unsafe patterns in the published content or in the build tooling), please open a [private security advisory](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) for this repository if that feature is enabled, or contact the repository owner through GitHub with enough detail to reproduce or assess the issue.

Do not open public issues for undisclosed security vulnerabilities.
