Two's Complement
How CPUs represent negative integers. Negate = flip all bits + 1. 8-bit range: -128 to 127. Overflow wraps silently — INT_MAX + 1 = INT_MIN. This is a real bug source.
IEEE 754 Float
32-bit: sign(1) + exponent(8) + mantissa(23). 0.1 + 0.2 ≠ 0.3. Never use floats for money. Use integer cents, Decimal, or BigDecimal. Infinity and NaN are valid float values.

// CACHE LOCALITY MATTERS

Sequential memory access (arrays) is 10–100x faster than random access (linked lists) because the CPU prefetcher can predict and pre-load the next cache line. Always prefer arrays over linked lists unless you truly need O(1) insert/delete.

CPU Architecture Essentials
Control Unit
Fetch → Decode → Execute cycle. Manages program counter. Modern CPUs pipeline, speculate, and reorder instructions out-of-order.
ALU
Arithmetic Logic Unit. Integer math and comparisons. FPU handles floating point. SIMD units process multiple values in parallel (AVX, NEON).
Virtual Memory
OS illusion: every process sees a full address space. MMU translates virtual → physical via page tables. Page fault = load from disk.
Interrupts
Hardware signals the CPU to pause and handle I/O, timer events, or exceptions. IRQ priority. Context save/restore on every interrupt.
